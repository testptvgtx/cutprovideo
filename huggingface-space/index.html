Open stack

Built with ONNX Runtime, WebGPU, WebAssembly, WebCodecs, and the Web Audio API.

Inside the editor

One workspace, from source media to export.

CC

Automatic captions

Whisper small q8 transcription with waveform-aware timing and editable, WYSIWYG subtitle styling.

VO

Browser voiceovers

Multilingual Piper/VITS voices and Kokoro English generation, lazy-loaded when needed.

ST

Source separation

Isolate vocals from the current source-audio clip and place the instrumental stem on the music track.

AV

Talking avatars

JoyVASA audio-to-motion with LivePortrait neural rendering on compatible WebGPU hardware.

CV

Vision-aware editing

Timestamped subject detection, smart crop, background removal, and caption avoidance for images and video.

EX

Browser-side export

Preview the composed result and export MP4 or WebM without handing the project to a render server.

Current limitations

Experimental, and honest about it.

  • Initial AI model downloads can be large.
  • Browser memory and WebGPU support vary by device.
  • 512 × 512 neural avatar rendering is still slow on current consumer GPUs.
  • This is not yet a replacement for a mature desktop NLE.
